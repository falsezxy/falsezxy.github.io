一、DNS

1.1 DNS是什么？什么作用？

DNS（Domain Name System）是互联网的一项服务，它作用于OSI七层模型的第7层（L7应用层）。DNS的核心功能是将域名解析为IP地址（翻译官），使用户可以通过易记的域名访问网络资源。

1.2 DNS记录类型—面试问

① A记录（主机记录）

将域名解析为IP地址

例如：example.com → 192.168.110.140

② P记录（指针记录）

反向解析IP到域名，将IP地址转换回对应的域名

例如：192.168.110.140 → example.com

1.3 DNS解析配置

配置文件 位置 主要用途 优先级
/etc/hosts 本地主机文件 内部IP-主机名/域名映射 最高
/etc/resolv.conf DNS解析配置 指定DNS服务器地址 中
/etc/sysconfig/network-scripts/ifcfg-ens33 网卡配置 网卡级别DNS配置 中

二、DHCP

DHCP是一种网络协议，用于动态分配IP地址和网络配置。当新设备加入网络时：

  1. 广播寻址 - 新设备发送DHCP DISCOVER报文，寻找DHCP服务器
  2. IP分配 - DHCP服务器从地址池中选择可用IP地址
  3. 地址确认 - 服务器回复DHCP OFFER报文，告知可分配的IP地址范围
  4. 租期设定 - 分配的IP地址有租期限制，通常为2小时

三、SSH

3.1 SSH是什么，什么作用？

SSH（Secure Shell）是一种远程连接工具，同时也是一种网络协议。它提供了安全的远程登录和文件传输功能，采用密码认证或密钥认证两种方式。

3.2 SSH远程登录

语法:ssh [-p 端口] 目标IP

注:首次连接需要输入目标主机的登录密码。

如果有在hosts文件中设置对应主机名的解析记录，也可以使用主机名连接

3.3 跨主机文件传输 scp

SSH进行跨主机传输文件是通过scp（ssh+copy）来实现的。

scp:scp 需要复制的文件路径 需要复制到的目标路径

参数说明：
root - 申明登录权限
@ - 连接符号
192.168.110.140 - 目标IP
:/opt - 目标主机的路径

⭐面试可能问：

内网服务器下很慢，怎么办？

借其他电脑如windows下载，再传过去(scp)。比如堡垒机

四、SSH密钥对的应用

密钥对认证原理

SSH密钥对认证相比密码认证更加安全，采用非对称加密算法，包含两个密钥：

公钥（Public Key）

  • 类比：门锁
  • 存放位置：需要发送给远程服务器
  • 文件名：id_rsa.pub
  • 作用：用于加密，公开分享

私钥（Private Key）

  • 类比：配套的钥匙
  • 存放位置：保存在本地，严格保密
  • 文件名：id_rsa
  • 作用：用于解密，仅自己持有
  • 权限：必须设置为600

密钥对生成与配置

① 生成RSA密钥对

执行成功后会生成两个文件：

  • /root/.ssh/id_rsa - 私钥
  • /root/.ssh/id_rsa.pub - 公钥

② 推送公钥到目标主机

语法：ssh-copy-id 目标IP

③ 免密登录

语法：ssh 目标主机名称

密钥对配置原理

当使用密钥对进行认证时，SSH连接过程如下：

  1. 客户端发起连接请求
  2. 服务器发送一个随机数字给客户端
  3. 客户端使用私钥对这个数字进行签名
  4. 服务器用公钥验证签名的有效性
  5. 验证成功则允许连接，无需输入密码

这种方式比密码认证更安全，因为私钥永远不会在网络上传输。

常见问题排查

Q：ssh-copy-id后仍需要密码？

  • 检查目标主机的 ~/.ssh/authorized_keys 文件是否存在公钥内容
  • 检查文件权限：authorized_keys 应为 600，.ssh 目录应为 700

认证方式 密码认证 密钥对认证
安全性 较低，易受暴力破解 高，密钥长度通常为2048位或更高
便利性 低，每次连接都要输入密码 高，配置后可免密登录
私密性 密码在网络传输时有泄露风险 私钥不在网络上传输，更安全
适用场景 临时连接、一次性使用 自动化脚本、频繁连接、生产环境

总结

DHCP - 自动分配IP地址，通过DISCOVER → OFFER → REQUEST → ACK四步完成，IP有租期限制。

DNS - 进行域名解析，优先级为：/etc/hosts（本地，优先级最高）→ /etc/resolv.conf（DNS服务器）→ 其他。/etc/hosts用于内网映射，/etc/resolv.conf用于外网查询。

SSH - 远程连接工具。密码认证需每次输入密码；密钥对认证（公钥+私钥）更安全且可免密登录，是生产环境首选。
