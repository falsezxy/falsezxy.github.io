Shell相关知识

一、Shell 基础概念

1.1 什么是 Shell？

Shell 是一种弱类型的小脚本语言，最简单的 Shell 脚本就是把多条命令组合在一起，一次性执行。

Shell 的核心作用：

  • 命令解释器：解释并执行用户输入的命令
  • 脚本执行器：批量执行预定义的命令序列

1.2 Shell 解释器类型

解释器 路径 说明
Bash /bin/bash CentOS 7 默认解释器
Sh /bin/sh CentOS 7 之前的默认解释器
Csh /bin/csh C 语言风格的 Shell

1.3 Shell 应用场景

  • ✅ 重复性操作自动化
  • ✅ 交互性任务处理
  • ✅ 批量事务处理
  • ✅ 服务运行状态监控（巡检）
  • ✅ 定时任务执行

9.1 case 语句基本概念

case 语句用于处理多个条件分支，类似于其他编程语言中的 switch 语句。当需要判断一个变量是否匹配多个固定值时，case 语句比 if-elif-else 更简洁高效。

9.2 case 语句语法

格式：

语法说明：

  • case：开始语句
  • in：关键字
  • ) ：每个值后面必须跟右括号
  • ;;：表示每个分支的结束（必须有）
  • *)：默认情况，类似于 else
  • esac：结束语句（case 反写）

⚠️ 注意：每个分支必须以 ;; 结尾，最后的 *) 也要有 ;;
